## Require environment
Need iPhone and mac connected by iCloud


## Step
1. get access db permission
```shell
sudo chmod 777 ~/Library/Messages/chat.db
```

2. set phone number at **config** file


3. run message-forward.sh
```shell
./message-forward.sh
```
